Change the Design of your profile page using CSS

Follow these instructions to modify your profile by changing styles in the CSS file.

  1. Open Visual Studio Code and choose File > Open to select your sites folder (if you did this before, it may already be open to that folder). Navigate to your profile page directory and open the index.html and main.css in edit mode.

  2. In the index.html, make sure you see this tag inside the head / head area: 
    <link rel="stylesheet" href="main.css">

  3. Go to the CSS file, and find the section for header.

  4. Change the background color for the header to blue, or any other primary color. Visual Studio Code will show you which colors you can pick.

  5. Find the section for img.

  6. Change the width to width: 50%; Add these lines: 
    
float: left;
    
padding: 10px;

  7. Find the section for body.

  8. Change the margin-left and margin-right to 10px;

  9. Back in the HTML, add a quote between paragraphs and surround it with these tags:
    <blockquote></blockquote>

  10. In the css, add this style for blockquote: 
    blockquote {
    
font-family: times;
    
font-size: 24px;
    
font-style: italic;
    
}


  11. Publish it: Go to your github account and copy and paste the new CSS into the main.css there and commit changes. A few minutes later, check to see if the style changes were made.

    12. More Enhanced CSS

  12. Experiment with float to control how elements sit alongside each other on the page. You already used float: left on your image — try applying it to other elements and see how the layout shifts.

  13. Add borders to give your elements more visual definition. Try adding these lines to your header CSS: 
    border: 1px solid #333;
border-radius: 12px;
    The border-radius property rounds the corners. Try increasing the value to see a more dramatic curve.

  14. Try text formatting, especially alignment. You can center text in a section using text-align: center;, or align it to the right with text-align: right;. Try applying this to your header or a paragraph.

  15. Explore margins and padding using the CSS box model. Remember: padding is the space inside an element's border, and margin is the space outside it. Try adjusting both on your header or body to see how they affect spacing.

  16. Add link styling to make your navigation or body links look more polished. Try adding these two new selectors to your CSS: 
    header a {text-decoration: none; color: white;}
header a:hover {color: red;}
    The first rule removes the underline and changes link color. The second changes the color when a user hovers over a link. Try swapping out the colors with your own choices.

  17. Add a background-image to your page. You can find free, high-quality images to use at https://unsplash.com/. Download an image, upload it to your project folder, and reference it in your CSS.

    To add the image with a subtle transparency effect so your text stays readable, replace your existing body background property with this: 
    body {
  background: linear-gradient(rgba(255, 255, 255, 0.5), rgba(255, 255, 255, 0.5)),
        url("URL TO IMAGE HERE");
  background-size: cover;
  background-position: center;
}
    Replace URL TO IMAGE HERE with the path to your image file (or a full URL). The rgba(255, 255, 255, 0.5) overlay adds a white wash at 50% opacity — adjust the last number (between 0 and 1) to make it more or less transparent.
