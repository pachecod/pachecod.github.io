1. Go to the following Github repository and copy the index.html file and main.css file above into two documents in Sublime Text. Save the files as index.html and main.css in a folder on your desktop.

    https://github.com/pachecod/pachecod.github.io/tree/master/htmlcsspractice

  2. Double click the index.html file to open it in a browser. Situate the sublime text and browser next to each other.

  3. Add this line to the index.html right before the title tags: 
    <link rel="stylesheet" href="main.css">
  4. After the body tag, create a header with <header></header> tags: 
    <header></header>

  5. Enter a title for your page inside the header tags:
    <title></title>

  6. Write a headline and surround it with <h1></h1> tags:
    <h1></h1>

  7. Write a paragraph and surround it with <p></p> tags.
    <p></p>

  8. Find a link to an image on the Internet, and surround it with <img src=""/> tags. Put that inside one of the paragraphs.

  9. Go to the CSS tab in Sublime Text, and find the section for header.

  10. Change the back-ground color to blue.

  11. Add this line under that: 
    color: white;

  12. Find the section for img.

  13. Change the width to width: 50%; Add these lines:
    float: left;
    padding: 10px;

  14. Find the section for body.

  15. Change the margin-left and margin-right to 10px;

  16. Back in the HTML, add a quote between paragraphs and surround it with these tags:
    
<blockquote></blockquote>

  17. In the css, add this style for blockquote:
    blockquote {
      font-family: times;
      font-size: 24px;
      font-style: italic;
    }
