Change the Design of your profile page using CSS

Follow these instructions to modify your profile by changing styles in the CSS file.

  1. Open Visual Studio Code and choose File > Open to select your sites folder (if you did this before, it may already be open to that folder). Navigate to your profile page directory and open the index.html and main.css in edit mode.

  2. In the index.html, make sure you see this tag inside the head / head area: 
    <link rel="stylesheet" href="main.css">

  3. Go to the CSS file, and find the section for header.

  4. Change the background color for the header to blue, or any other primary color. Visual Studio Code will show you which colors you can pick.

  5. Find the section for img.

  6. Change the width to width: 50%; Add these lines: 
    
float: left;
    
padding: 10px;

  7. Find the section for body.

  8. Change the margin-left and margin-right to 10px;

  9. Back in the HTML, add a quote between paragraphs and surround it with these tags:
    <blockquote></blockquote>

  10. In the css, add this style for blockquote: 
    blockquote {
    
font-family: times;
    
font-size: 24px;
    
font-style: italic;
    
}


  11. Publish it: Go to your github account and copy and paste the new CSS into the main.css there and commit changes. A few minutes later, check to see if the style changes were made.
