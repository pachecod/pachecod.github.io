01 · Images

Responsive Image img-fluid

Add .img-fluid to any <img> so it scales down on small screens but never grows beyond its natural size. Combine with .rounded or .rounded-circle for shape variants.

HTML 
<!-- Basic responsive image -->
<img src="your-image.jpg" class="img-fluid rounded" alt="Description"/>

<!-- Circle crop (works best on square images) -->
<img src="your-image.jpg" class="img-fluid rounded-circle" alt="Description"/>

<!-- Thumbnail (adds a subtle border + padding) -->
<img src="your-image.jpg" class="img-fluid img-thumbnail" alt="Description"/>

03 · Alerts

Alerts

Contextual feedback messages for the user. Add data-bs-dismiss="alert" and the .alert-dismissible class to make them closeable.

📰 Tip: This is a primary alert — use it for general info.
✅ Success! Your story has been published.
⚠️ Warning: Your draft auto-saves every 60 seconds.
🚫 Error: Could not connect to the database.
ℹ️ Dismissible: Click the × to close this alert.
HTML 
<!-- Static alerts -->
<div class="alert alert-primary">Primary alert</div>
<div class="alert alert-success">Success alert</div>
<div class="alert alert-warning">Warning alert</div>
<div class="alert alert-danger">Danger alert</div>

<!-- Dismissible alert (requires Bootstrap JS) -->
<div class="alert alert-info alert-dismissible fade show" role="alert">
  Your message here.
  <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>

06 · Accordion

Accordion

Collapsible sections — perfect for FAQs, glossaries, or multi-step instructions. Only one panel is open at a time by default.

Bootstrap is an open-source CSS framework developed by Twitter. It includes pre-built components, a responsive grid system, and JavaScript plugins that help you build websites faster without writing everything from scratch.

Basic CSS knowledge helps, but Bootstrap is designed so you can get far with just HTML and the right class names. As you get more comfortable, CSS lets you customize Bootstrap's defaults.

The simplest way is to paste the Bootstrap CDN <link> tag into your <head> and the <script> tag before your closing </body>. No installation required.

Yes — Bootstrap is mobile-first by design. Its responsive grid collapses from multi-column desktop layouts to single-column stacks on small screens automatically.
HTML 
<div class="accordion" id="myAccordion">

  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button" type="button"
              data-bs-toggle="collapse" data-bs-target="#item1">
        Question One
      </button>
    </h2>
    <div id="item1" class="accordion-collapse collapse show"
         data-bs-parent="#myAccordion">
      <div class="accordion-body">
        Answer goes here.
      </div>
    </div>
  </div>

  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button collapsed" type="button"
              data-bs-toggle="collapse" data-bs-target="#item2">
        Question Two
      </button>
    </h2>
    <div id="item2" class="accordion-collapse collapse"
         data-bs-parent="#myAccordion">
      <div class="accordion-body">
        Answer goes here.
      </div>
    </div>
  </div>

</div>
07 · Carousel

Carousel

A slideshow for cycling through images or content. Add data-bs-ride="carousel" to auto-play, or omit it to require user interaction. Captions go in .carousel-caption.

Scene One

Captions describe what's in the image for readers who rely on them.

Scene Two

Each slide can have its own headline and caption text.

Scene Three

The overlay text sits on a semi-transparent gradient automatically.

Scene Four

Add as many slides as you need — just copy the .carousel-item block.

HTML 
<div id="myCarousel" class="carousel slide" data-bs-ride="carousel">

  <!-- Dot indicators -->
  <div class="carousel-indicators">
    <button type="button" data-bs-target="#myCarousel" data-bs-slide-to="0" class="active"></button>
    <button type="button" data-bs-target="#myCarousel" data-bs-slide-to="1"></button>
    <button type="button" data-bs-target="#myCarousel" data-bs-slide-to="2"></button>
  </div>

  <!-- Slides -->
  <div class="carousel-inner">

    <div class="carousel-item active">
      <img src="slide1.jpg" class="d-block w-100" alt="Slide 1"/>
      <div class="carousel-caption d-none d-md-block">
        <h5>Caption Headline</h5>
        <p>Supporting text here.</p>
      </div>
    </div>

    <div class="carousel-item">
      <img src="slide2.jpg" class="d-block w-100" alt="Slide 2"/>
    </div>

  </div>

  <!-- Prev / Next controls -->
  <button class="carousel-control-prev" type="button"
          data-bs-target="#myCarousel" data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
    <span class="visually-hidden">Previous</span>
  </button>
  <button class="carousel-control-next" type="button"
          data-bs-target="#myCarousel" data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
    <span class="visually-hidden">Next</span>
  </button>

</div>
08 · Modal

Modal Dialog

Overlays that appear on top of the page. Good for confirmation dialogs, image lightboxes, sign-up forms, or quick previews. Triggered by a data-bs-toggle="modal" attribute on any element.

Modal Headline

This content appears in an overlay on top of your page. You can put anything here — text, forms, images, or entire components.

Clicking the backdrop or pressing Esc also closes the modal.

Caption: This is an example lightbox image.

HTML 
<!-- Trigger button -->
<button type="button" class="btn btn-primary"
        data-bs-toggle="modal" data-bs-target="#myModal">
  Open Modal
</button>

<!-- Modal markup (put anywhere in body) -->
<div class="modal fade" id="myModal" tabindex="-1" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">

      <div class="modal-header">
        <h5 class="modal-title">Title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>

      <div class="modal-body">
        Your content here.
      </div>

      <div class="modal-footer">
        <button class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
        <button class="btn btn-primary">Save Changes</button>
      </div>

    </div>
  </div>
</div>

12 · Tooltips & Popovers

Tooltips & Popovers

Tooltips show a single line on hover; popovers show a larger box with a title and body. Both require Bootstrap JS and must be activated manually with a small script (shown in the code block below).

HTML + JS 
<!-- Add title="" and data-bs-toggle="tooltip" to any element -->
<button class="btn btn-secondary"
        data-bs-toggle="tooltip"
        data-bs-placement="top"
        title="Your tooltip text here">
  Hover me
</button>

<!-- Popover (click to open) -->
<button class="btn btn-info"
        data-bs-toggle="popover"
        data-bs-title="Popover Title"
        data-bs-content="Longer body text here.">
  Click me
</button>

<!-- REQUIRED: activate all tooltips and popovers with JS -->
<script>
  // Tooltips
  document.querySelectorAll('[data-bs-toggle="tooltip"]').forEach(el => {
    new bootstrap.Tooltip(el);
  });
  // Popovers
  document.querySelectorAll('[data-bs-toggle="popover"]').forEach(el => {
    new bootstrap.Popover(el);
  });
</script>
Starter Template

Full Page Boilerplate

Copy this skeleton into a new .html file to get Bootstrap 5 — CSS, JS, and Icons — all loaded from CDN with no installation.

HTML — Full Starter Template 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8"/>
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>My Bootstrap Page</title>

  <!-- Bootstrap 5 CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
        rel="stylesheet"/>

  <!-- Bootstrap Icons (optional) -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css"
        rel="stylesheet"/>
</head>
<body>

  <div class="container mt-4">
    <h1>Hello, Bootstrap!</h1>
    <p>Start building here.</p>
  </div>

  <!-- Bootstrap JS bundle (includes Popper) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

  <!-- Activate tooltips & popovers -->
  <script>
    document.querySelectorAll('[data-bs-toggle="tooltip"]')
      .forEach(el => new bootstrap.Tooltip(el));
    document.querySelectorAll('[data-bs-toggle="popover"]')
      .forEach(el => new bootstrap.Popover(el));
  </script>

</body>
</html>
