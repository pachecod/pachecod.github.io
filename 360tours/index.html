What It Does

This point and click tool creates 360 tours that can be exported and hosted anywhere, such as on Github Pages. You can upload 360 photos or add links to 360s elsewhere, and then link them together with hotspots.

You can also annotate the tours with content including 2D photos, audio, text overlays, and even links to any web site. The web link feature can be used to tie multiple tours together, or to simply send people to an external site.

When you are finally done, there are two ways to save your tour:

  • Complete Project: Click Save and choose this option to download a folder that you can upload to something like Github Pages.
  • Submit to Professor: This sends the tour to the Professor Dashboard.

The Professor Dashboard

This is a page that anyone is able to access, so don't be turned off by the professor part! Try using "Send to Professor," then go to the dashboard to see your submitted files.

Anyone with access to this page can export from there, or click the Host button to create a link for a quick online preview. Please don't use the hosted link publicly, as that could cause Render.com costs to go up. It's best to just do a quick preview, and then if you are satisfied you can export the full site and host it elsewhere.
